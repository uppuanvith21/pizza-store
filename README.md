# Pizza Store Website 🍕

A simple and responsive Pizza Store website developed using Python. This project allows users to browse pizza varieties, view prices, place orders, and explore store information through an interactive interface.

## Features

* 🍕 Display different pizza categories and menu items
* 🛒 Online pizza ordering functionality
* 💰 Price listing for all pizzas
* 📱 Responsive and user-friendly design
* 📍 Store information and contact details
* 🔍 Easy navigation and menu browsing

## Technologies Used

* Python
* HTML
* CSS
* JavaScript
* SQLite (if database is used)

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd PizzaStore
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```text
http://localhost:5000
```
